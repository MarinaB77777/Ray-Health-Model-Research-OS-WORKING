 Question Bank Architecture v1



Идентификация вопроса

Каждый вопрос имеет два идентификатора:
question_id
и
code
Пример:
question_id = UUID0001
code = KCog1




Роли идентификаторов

question_id
Постоянный технический идентификатор.
Никогда не меняется.
Используется в БД, логах, аналитике, версиях вопросов.
Даже если вопрос будет переписан.



code
Научный / модельный идентификатор.
Используется в формулах и расчётах.
Пример:
KCog1
MG4
PR2
Q6




Версии

Допускаются разные версии одного конструкта.

Пример:
UUID0001 → KCog1 v1
UUID0157 → KCog1 v2
UUID0421 → KCog1 adaptive
Все относятся к:
code = KCog1




Локализация

Один вопрос:
question_id = UUID0001
code = KCog1
может иметь:
ru
en
es
локализации.

Локализация не создаёт новый вопрос.




Источник правды

Единственный источник вопросов:
model_engine/question_banks/ru.py
(позже)
question_banks/en.py
question_banks/es.py




Разделение

Банк вопросов:
что существует
Логика опроса:
что спрашивать
в каком порядке
при каких условиях
Это разные сущности.




Нормализация кодов

Код вопроса:
KCog1
MG3
PR5
фиксируется навсегда.

Номер вопроса в анкете может меняться.

Порядок показа может меняться.

Логика опроса может меняться.

Код вопроса не меняется.
