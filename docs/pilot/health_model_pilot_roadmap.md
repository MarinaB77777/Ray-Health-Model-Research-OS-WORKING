# 6. Constitutional Invariants

## External Core / Base Ray Invariants

1. No fabrication  
No data → no invention.

2. NOT_ENOUGH_DATA = valid state  
Insufficient data is a legitimate operational result.

3. Unknown ≠ 0  
Unknown values are not treated as normal/default.

4. Prediction ≠ truth  
Forecasts remain probabilistic.

5. Recommendation ≠ authority  
Recommendations do not grant execution rights.

6. Understanding ≠ authority  
Better understanding does not expand permissions.

7. Runtime ≠ truth engine  
Runtime coordinates but does not define reality.

8. Governance ≠ reasoning  
Governance checks boundaries, not truth.

9. Memory ≠ authority  
Remembered information is not automatic permission.

10. Human Profile ≠ identity  
Profiles are operational approximations, not identity truth.

11. Base Ray ≠ Inner Core  
Base Ray operates separately from Inner Core.

12. Inner Core excluded from Pilot v1  
No deep priorities or pain points are used.

13. Projection Layer excluded from Pilot v1  
No projection weights or deep harm minimization.

14. External AI ≠ Ray truth  
External AI remains a bounded tool only.

15. External AI agreement ≠ validation  
Multiple AI agreement is not proof of truth.

16. Visibility ≠ authority  
Seeing information does not grant action rights.

17. Access ≠ permission  
Technical access does not equal allowed use.

18. Capability ≠ permission  
Ability to perform an action does not legitimize it.

19. WAITING ≠ answered  
Waiting for data is not equivalent to receiving data.

20. Acquisition request ≠ acquisition result  
Requesting information is not information itself.

21. Silence ≠ confirmation  
Lack of response is not consent.

22. Contradiction ≠ failure  
Contradiction triggers clarification, not fabrication.

23. Unresolved truth > fake certainty  
Honest uncertainty is preferable to false confidence.
