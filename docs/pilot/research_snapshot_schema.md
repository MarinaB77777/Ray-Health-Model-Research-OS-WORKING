# Research Snapshot Schema

## Purpose

This document defines the bounded constitutional schema semantics for research snapshots used by the Research Analysis Layer inside the university pilot.

The purpose of this schema is:
- bounded research ingestion;
- uncertainty-aware research exports;
- reproducible scientific analysis;
- explainable snapshot semantics;
- anti-hidden-authority protection;
- anti-ontology drift protection.

This document defines:
- snapshot semantic structure;
- bounded export categories;
- uncertainty-preserving schema semantics;
- research-safe snapshot boundaries;
- bounded research interpretation structures.

This document does NOT:
- define Runtime authority;
- define participant truth;
- define unrestricted longitudinal participant models;
- define unrestricted behavioral ontologies;
- define unrestricted analytics infrastructure;
- define implementation payload schemas;
- define database schemas.

---
