# Research Laboratory Layout Contract v1

## Purpose

This document defines the logical layout of the Research Laboratory.

The layout is independent from frontend implementation.

Desktop, tablet and web interfaces use the same logical structure.

---

# General Principle

The Research Laboratory is a workspace.

It is not a menu.

Everything is organized around scientific work.

---

# Main Workspace

The workspace consists of five permanent zones.

--------------------------------------------------

Navigation

↓

Workspace

↓

Inspector

↓

Activity

↓

Status

--------------------------------------------------

Every module follows this layout.
