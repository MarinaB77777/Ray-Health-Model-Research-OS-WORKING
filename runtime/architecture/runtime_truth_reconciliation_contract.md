≠ correctness

- unquestionable interpretation.

## Contradiction Legitimacy Boundary

Contradiction is architecturally legitimate.

Core invariant:

unresolved contradiction  
≠ invalid architecture

- incomplete;
- context-limited;
- causally ambiguous.

## Contradiction Confidence Boundary

Possible contradiction is not verified contradiction.

Core invariants:

possible contradiction  
≠ verified contradiction

absence of detected contradiction  
≠ verified consistency

Runtime may detect possible inconsistency between:
- human self-report;
- behavior;
- biosignals;
- memory;
- context;
- external sources;
- social signals;
- calendar/location/activity data.

Possible contradiction should trigger:
- evidence quality review;
- source consistency review;
- weight recalibration where appropriate;
- context investigation;
- human clarification;
- reliable external verification where allowed.

Runtime must not:
- hide detection errors;
- accuse by default;
- fabricate reconciliation;
- make one detection error the central truth of the situation.

Reconciliation errors require correction, not concealment.

Uncertainty detection requires investigation, not fabrication.

## Source Weighting Boundary

No source is globally absolute.

Core invariant:

source authority  
≠ universal truth authority

Source confidence must remain:
- contextual;
- calibrated;
- freshness-aware;
- signal-quality-aware;
- artifact-aware;
- domain-specific.

A calibrated stable sensor may carry strong evidential weight when:
- calibration is valid;
- operating conditions are unchanged;
- signal quality is acceptable;
- artifact risk is low;
- context supports interpretation.

Core invariants:

calibrated stable sensor shift  
requires context investigation

high-confidence sensor interpretation  
≠ complete human-state understanding

Runtime must not dismiss calibrated sensor change merely because human self-report says “everything is normal.”

Human self-report remains important.

Sensor evidence remains important.

Neither source becomes absolute truth by itself.
